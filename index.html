Mon programme d'apprentissage

  • Comprendre le fonctionnement d'un ordinateur et d'un réseau local

Un ordinateur (matériel) est composé principalement de :

  • Une carte mère (MotherBoard)
  • Un processeur (CPU)
  • De la mémoire vive (RAM)
  • Un disque dur (SSD)
  • Une carte gaphique (GPU)
  • Un BIOS (Basic Input Output System)
  • Une carte réseau (Ethernet, Wifi)
  • Des périphériques d'entrée/sortie (E/S) comme le clavier ou l'imprimante

  • Comprendre le système d'exploitation

Les deux systèmes d'exploitations (OS: Operating System) les plus utilisés en entreprise sont Linux et Windows Server.
Le système d'exploitation est une couche logicielle qui fait l'interface entre les applications et le matériel.
Le système d'exploitation est composé d'un noyau (kernel), de pilotes (drivers), d'outils système, de bibliothèques, de services.
Lorsque j'allume l'ordinateur:

  • Le BIOS s'allume en premier (il prépare le matériel: CPU, RAM, SSD GPU, ...)
  • Puis le BIOS lance le bootloader qui va charger le système d'exploitation
